1. f(n) represents the total number of options of the break, then f(n) = 2^(n-1)

2. g(n) represents the total number of options with at least one 2 (TWO) as its item,
   we have,
      1. d[n] = g[n] - 2*g[n-1]
      2. d[n] = d[n+4] - d[n+3] - d[n+2]
   hence,  
      g(n) = 3*g(n-1) - g(n-2) - 2*g(n-3) + g(n-4) - 2*g(n-5)
    
3. Let t(n) represent the number of twopals of number n,
	if n is ODD number, t(n) = t(n-1) - f(n/2) + g(n/2);
	if n is EVEN number, t(n) = t(n-1) - g(n/2) + f(n/2)
